### 🌟 Featured Projects

#### 🩺 [RAGnosis — AI-Powered Medical Report Analysis](https://github.com/rudranis/RAGnosis-AI-Powered-Medical-Report-Analysis)
AI-driven medical report analysis and hospital management platform — turns raw lab reports into plain-English insights and supports Patient/Doctor/Receptionist workflows.
- **Tech:** React, Flask, MongoDB, BioBERT, BART, FAISS, Groq LLaMA 3
- **Focus:** Applied AI, RAG architecture, full-stack integration

#### 🎫 [PICT Guard — Smart Gate Pass Management System](https://github.com/rudranis/PICT-Guard-Smart-Gate-Pass-Management-System)
QR-code based gate pass system for college campuses with role-based portals for Admin, Student, Faculty, Guard, and Visitor — handling secure entry/exit validation end to end.
- **Tech:** HTML, CSS, Java, React, FastAPI/Flask, MongoDB, AWS
- **Live:** [pict-guard-smart-gate-pass-manageme-lime.vercel.app](https://pict-guard-smart-gate-pass-manageme-lime.vercel.app/)
- **Focus:** System design, role-based access control, real-world deployment

#### 🧾 [Automated AWS Receipt Processing System](https://github.com/rudranis/Automated_AWS_ReceiptProcessingSystem)
Fully serverless pipeline that digitizes receipts automatically — from upload to structured storage to email confirmation, with zero servers to manage.
- **Tech:** AWS S3, Lambda, Textract, DynamoDB, SES
- **Focus:** Serverless architecture, event-driven AWS orchestration

#### 🎙️ [VoiceVault — Secure Voice Recording App](https://github.com/rudranis/VoiceVault)
Browser-based voice recording app with authenticated, secure cloud storage of recordings.
- **Tech:** React, AWS Amplify, Cognito, S3
- **Focus:** Cloud auth flows, secure media storage

#### 💬 [Spring Boot Chat App](https://github.com/rudranis/springboot-chat-app-project)
Real-time chat application built on Spring Boot, exploring WebSocket-based communication patterns.
- **Tech:** Java, Spring Boot, WebSocket
- **Focus:** Real-time communication, backend messaging design

#### 🏙️ [Citizen Feedback Management System](https://github.com/rudranis/citizen-feedback-management-mongodb)
Civic platform where citizens submit, track, and follow up on feedback and complaints through structured, MongoDB-backed workflows.
